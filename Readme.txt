Simple action adventure 3D game. Aim of this project was to get familiar with as many subjects connected with game development as possible in a limited amount of time. 

For this project I used UnrealEngine v4.22 + Blueprints, Blender v2.79 and C++ for coding.

Below there is list of folders which contains assets that were not created by me.
